FHIR serializer versions follow schema "fhir-r4-obs-dr-v1" where

- "r4" indicates FHIR release. Currently R4 is most commonly used. R5 is 
  available. R6 is in works.
- "dr-obs" indicates FHIR resources: DiagnosticReport and Observation, 
  respectively.
- "v1" is a version of the FHIR serializer given the specified FHIR release
  version and the FHIR resources. Create a new version, when changing behavior
  of the FHIR serializer (such as changing field mapping, logic).
